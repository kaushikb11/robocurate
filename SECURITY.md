# Security Policy

## Supported versions

RoboCurate is in pre-alpha. Only the latest `0.0.x` release is supported; there are no
security backports to earlier releases.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a vulnerability

Please report security vulnerabilities **privately** through GitHub Security Advisories,
using the **"Report a vulnerability"** button on the repository's **Security** tab. This
keeps the report confidential while it's being investigated and fixed.

Please do not open a public issue for security reports. We'll acknowledge your report and
keep you updated on the fix and disclosure timeline.
